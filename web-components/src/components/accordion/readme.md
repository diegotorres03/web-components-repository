# Accordion Component

**Tag:** `app-accordion`


## Samples:

```html
<app-accordion>
  <section>
    <h5>Game instructions</h5>
    <p>This is a memory game, where you have to find pairs in a set of cards.</p>
    <p>You will have X minutes to solve multiple challenges and the score will be based on how far you get.</p>
  </section>
  <section>
    <h5>FAQs</h5>
    <b>Question 1:</b>
    <p>Answer 1</p>
    <hr>
    <b>Question 2:</b>
    <p>Answer 2</p>
    <hr>
  </section>
</app-accordion>
```
